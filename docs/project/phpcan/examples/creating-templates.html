Creating templates

The base template

The first template we have to do is the "base" template which load the rest of the templates. This template has to be saved in the templates folder as "html.php", because this is the name we have put in the routes configuration.

Example of base template

<html>
	<head>
		<title>My site title</title>
		<meta charset="utf-8" />

		<?php echo $Html->cssLinks($Config->templates['css']); ?>
	</head>

	<body>
		<h1>My Site</h1>

		<?php $Templates->render('content'); ?>
	</body>
</html>

We can access to the template configuration with $Config->templates array. We can print the css link code using the html function $Html->links and passing the array of css to include. In the body of the page, we call to the template "content" of the current route.

Creating other templates

Now we can create the templates for each route and this template will be loaded in the body of our base template.

This is the index template, saved as "content-index.php":

<h2>Welcome to my site</h2>

<p>You can find a lot of interesting articles here!!</p>

This is the about-us template, saved as "content-about-us.php":

<h2>About us</h2>

<p>We are Berto and María.</p>

This is the undefined template, saved as "content-404.php":

<h2>Page not found</h2>

<p>Sorry, whatever you are looking for, it isn't here. Try google :)</p>

This is the undefined template, saved as "content-articles.php":

<h2>Articles</h2>

<p>Enjoy</p>

Creating the articles

As our site can't use databases, we have to save the articles in files. A way to do that is saving each article in a file starting with "article-" and the name of the article, for example: "article-the-cows-in-galicia.php". These files will be saved in the templates folder and we can use the variable "article_file" defined in routes to know which article have to load and include it in the articles template.

In this example, we get the template file and check if exists. If the file exists, include it, otherwhise redirect to undefined page.

<h2>Articles</h2>

<p>Enjoy</p>

<?php
$template_file = 'article-'.$Vars->get('article_file').'.php';

if ($Templates->exists($template_file)) {
	$Templates->render($template_file);
} else {
	redirect(path('undefined'));
}
?>

This is an example of article, saved in the file "article-the-cows-in-galicia.php" in the templates folder.

<h1>The cows in Galicia</h1>

<p>There is a lot of cows in Galicia, so there is a lot of milk in Galicia.</p>

Now, you can read this beautiful article in the url "static/articles/the-cows-in-galicia" :)
