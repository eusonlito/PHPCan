Vars::getLanguage()

phpCan 1.0

Description

Returns the language of the current page. If param $language is defined, compares it with current language and returns true if both are equal or false if not.

Vars::getLanguage ([string $language])

language (string)
Language to compare.
//If the language is 'en'

//Return the current language
echo $Vars->getLanguage(); //en

//Compare with other languages
$Vars->getLanguage('en'); //true
$Vars->getLanguage('gl'); //false
