getImageObject()

phpCan 1.0

Description

Returns a new Image object.

getImageObject ()

What do this function does?

In phpCan there are various image proccesing classes (GD and Imagik for example). This function will return an inicializated object based in the configuration of $Config->images.

Dependences

This function needs the following classes to work:

  • config (as $Config->)
  • \files\images\Imagick or \files\images\GD
$Image = getImageObject();

$Image->load('my-image.png')->resize(400, 500)->save();
