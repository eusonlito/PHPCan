Usage

Once content module is installed and configurated, the user can login to manage the website content.

All tables are visible in the home and you can list, modify, create and delete their registers. You can also manage the relations between the registers. For example, editing a post, you can see all comments, tags and categories related, and can relate more registers or unrelate them.
