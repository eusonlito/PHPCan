The content module

This is a powerful module to manage the content of the website. It's based of the database configuration (tables and formats) building the forms automatically. You can also see, upload, download and delete files and folder in the upload folder.

You can configurate some things like the label of the tables and formats and a little description. You can also ignore some tables and fields if you don't want make them accessible from this module. There is also a simple login system so you can create different configurations for each user

This module is built using html5 and css3 so only can be accessed using modern browsers (firefox and chrome).
