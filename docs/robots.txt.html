Research + Development + Sharing

This is a public repository of our open source projects. They have been developed in order to help us to build better websites, improve our internet experience, learn about new stuff or simply to enjoy ourselves. Most of these I+D+C projects are in early beta, but we hope that you can find them useful somehow.

  • $.ansSlider

    jQuery plugin to create sliders of everything

  • $media

    jQuery based library to manage html5 elements of video and audio

  • Bookmarklets

    Simple framework to build bookmarklets easily

  • Cache

    PHP Cache Interface with support for APC, Memcache, Memcached and files

  • cSans

    cSans is a light, flexible and modern CSS framework, compatible with modern browsers such as IE8+, Firefox, Safari, Opera... & other platforms

  • Imagecow

    PHP library to manipulate and generate responsive images

  • phpCan

    PHP 5 framework

  • styleCow

    Css preprocesor written in php
